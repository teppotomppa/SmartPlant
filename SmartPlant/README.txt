Kun avaat Arduino IDE koodit sovelluksessa, täytyy todennäköisesti luoda uusi sketchbook.
*****************************************************************************************
Arduino IDE ehkä tarjoaa sitä, kun koitat avata jonkin tiedoston.
****************************************************************
Muista avata kaikki tiedostot, ja lisätä ne samaan Sketchbookkiin, jotta toimii.
********************************************************************************
Kun avaat API:n, niin avaa ensin VSCode --> Open folder ja etsi API 0.1
***********************************************************************

Ja ainiin, Arduino IDE koodissa on Wi-Fi credentials kohta. Lisää siihen haluamasi Wi-Fi nimi
ja salasana, muuten et pääse Arduinolla samaan verkkoon, jossa servu on päällä.
